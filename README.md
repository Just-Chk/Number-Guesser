# Number Guessing Game

A simple and interactive number guessing game built with HTML, CSS, and JavaScript.

## Features
**Interactive Gameplay**: Guess numbers between 1 and 20
**Smart Hints**: Get feedback on how close your guess is
**Attempt Counter**: Track how many guesses you make
**New Game Button**: Restart the game anytime
**Clean UI**: Responsive design with visual feedback
**Keyboard Support**: Press Enter to submit guesses

## How to Play
1. Enter a number between 1 and 20 in the input field
2. Click "Guess" or press Enter to submit
3. Receive feedback (too high/too low)
4. Use hints to guide your next guess
5. Try to find the secret number in the fewest attempts!
